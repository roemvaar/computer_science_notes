# Designing Algorithms

You can choose from a wide range of algorithm design techniques.

* Sliding Window
* Divide-and-conquer

* Top K Elements
* BFS
* DFS
* Backtracking
* Dynamic Programming
    * Tabulation
    * Memoization
* Binary search
* Binary tree traversals
